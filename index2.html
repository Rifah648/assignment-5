Question-1: What is the different ways to select on element in the Dom?

    Answer: There are several ways to select elements in the DOM.

  • 1. Select an element by Id: const element = document.getElementById("myElement")
  • 2. Selects elements by ClassName: const elements = document.getElementsByClassName("myClass")
  • 3. Selects elements by tagName: const elements = document.getElementsByTagName("div")
  • 4. Select the first element that matches CSS selector : const element = document.querySelector(".myClass");
  • 5. Selects all elements that match CSS selector: const elements = document.querySelectorAll(".myClass");
  • 6. Select elements with specific attributes: const element = document.querySelector('[data-id="123]')

Question-3: What is event Delegation in the DOM?

    Answer: Event Delegation is a tecnique where a single event listener is added to a parent element instead of multiple child elements. This works because of event Bubbling, where events triggered on child elements propagate up to their ancestors.

Question-4: What is event bubbling in the DOM?

    Answer: Event bubbling is a concept in the DOM where an event, such as a click or a keypress, is triggered on an element and then propagates or "bubbles up" to its parent element in the DOM hierarchy. It works: When an event is triggered on a child element, it first runs the event handler on that child element. Then, the event moves up to its parent element, where the event handler is executed, and continues bubbling up through all the ancestor elements, one by one, until it resches the top-most ancestor.

Question-5: How do you create, add, and remove elements using JavaScript?

    Answer: To create, add, and remove elements using JavaScript:

  • To create an element, use document.createElement()method: let newElement = document.createElement('div');
  • To add the newly created element to the DOM, using appendChild() or append(): document.body.appendChild(newElement);
  • To remove an element, use removeChild() or remove(). let elementToRemove = document.getElementById('elementId'); elementToRemove.remove(); let parent = document.getElementById('parent'); let child = document.getElementById('child'); parent.removeChild(child);
